# Leaf Lens

The mobile runner app for the Leaf ecosystem.

## Features

- **QR Code Scanner:** Instantly preview your Leaf apps by scanning the QR code generated by `leaf lens`.
- **History:** Quickly access recently scanned URLs.
- **Native Plugins:** Pre-installed with common Capacitor plugins (Camera, Haptics, etc.).

## Development

1.  **Install Dependencies:**

    ```bash
    npm install
    ```

2.  **Run Locally (Web):**

    ```bash
    npm run dev
    ```

3.  **Run on Device:**
    ```bash
    npx cap run android
    # or
    npx cap run ios
    ```
